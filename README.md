# LockNotes
## What is LockNotes?
LockNotes came from my fear of not remember my right seat number - I have seen it a lot when traveling for work folks sitting in the wrong seat on flights.

I wanted a quick way without unlock my phone and waiting for my plane ticket app to refresh or digging for a screenshot. 

I thought it would be nice if on my locked screen I could see notifications with my seat # or gate # or any other info I wanted to add.

That is when LockNote was born.

Basically - Add notes to be displayed as notifications for quick view on your lock screen page.

Hope you enjoy!
